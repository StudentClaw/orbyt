## Emerging Study Strategies

- Ask for one concrete worked example first when material feels abstract or definition-heavy, then build the rule from there.
- On problems with multiple cases, work through one representative branch first, then map it back to the general pattern.
- For larger assignments, outline the work in stages up front so it moves cleanly from planning to building to refinement.
- Reserve the 7-9 AM pre-class block for the hardest problem solving, especially calculus.
- Before starting work in Canvas, check Announcements first, then verify the submission tab before uploading.
- Block fixed work hours before laying out study sessions.
